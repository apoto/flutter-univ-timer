# flutter_univ_timer

Flutter 大学 タイマーアプリ

* 分、秒、ミリ秒を表示しています
* 分、秒、ミリ秒がそれぞれ常に2桁表示されるよう、String型に変換し、padLeftメソッドを使用しました
* 3分経過で、画面遷移します
